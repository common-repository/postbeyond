Official PostBeyond Employee Advocacy platform integration plugin. Connect your PostBeyond account and push content to your hub right from WordPress

== Installation ==

If using .zip then

1. Extract `wp-postbeyond.zip`
2. Upload `postbeyond` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
